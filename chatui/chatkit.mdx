## About ChatKit
ChatKit is a ChatGPT-based application designed to deliver an intelligent and intuitive chat experience...[*read more*](https://chatkit.app/faq#what-is-chatkit).

## Prerequisites
Before you begin, ensure you have the following:  
1. API key from ClashAI

<Steps>
  <Step title="Creating api key">
  For this, you need to join the [Discord server](https://discord.gg/yyfGnyKj9j) and go to the [Dashboard channel](https://discord.com/channels/1231339329435996280/1302343342746505347).

  On the left sidebar, find the section labeled **"Manage API Key".** Here, you’ll click on **"Generate"** where your unique API key awaits—ready to unlock your explorations.
  </Step>
  <Step title="Setuping ChatKit">
  <img height="200" src="/images/chatkit.png" />
  In the upper right corner click on the settings icon, then go to the “Models” section, and then enter the same as in the photo above, in the “API key” field, you need to insert the key that we got in the first step.
  </Step>
</Steps>
